This folder contains page components for the Nutrition Connect app.
Pages are wired via src/routes/Router.jsx and protected through AuthContext.

Available pages:
- Login.jsx
- Register.jsx
- DashboardClient.jsx
- DashboardCoach.jsx
- DashboardAdmin.jsx
- Onboarding.jsx
- Plans.jsx
- Libraries.jsx
- Chat.jsx
- Tracking.jsx
- Billing.jsx
- Settings.jsx
